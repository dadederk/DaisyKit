NIMAS Miscellaneous Items - Image Quality

DAISY 3 Structure Guidelines
Last Revised: June 4, 2008

This page is initially intended for discussion. This section will be expanded as resolution is reached.

NIMAS requires the use of 300 DPI images, with the following image formats recommended: SVG, PNG, and JPG. There has been much debate regarding file sizes of PNG and JPG images, and as such, the following samples have been added for discussion.
