Appendix 2: DAISY DTB Announcement Scripts

DAISY 3 Structure Guidelines
Last Revised: June 4, 2008

All DAISY DTBs should contain a section that provides information about the DAISY book and how it can be read by the end user. This section may contain a general description of the structure of the book, number of levels, navigation features, etc. (See the Part 1 of the "DAISY Structure Guidelines").

Organizations normally also include production notes stating the name of the producer, name of narrator (if applicable), information regarding usage and copyright restrictions, and so on. To facilitate international exchange of materials thus reducing duplication of effort and waste of resources, it is important that each organization place this information in the same location in each book, within the first level heading, following title author and print copyright information. References to information such as other books in the series available from the producing organization, book clubs people may join, newsletters they may register for, etc. should be avoided as this can be confusing for a reader who is a library client with another organization in another country. It is recommended that information specific to the producing organization not be placed at the end of the book.

It is also recommended that producers place the description of how the DAISY book can be used in the same relative location in every DAISY DTB so that it is easy for the readers to find information about how to use the book. It may, for example, be placed immediately following the information about the producing organization. There should be a distinct heading for this information to facilitate easy access by the reader. Two possible options for the wording of this heading are: "How to use this DAISY Digital Talking Book" and "About this DAISY Digital Talking Book".

Script

The following information should be included in the description. Enter the relevant details and select the descriptions which best suit each DAISY DTB.

Number of CDs

  • Number of CDs: "This book is contained on __ CD(s)"
  • CD Number if the book spans more than 1 CD: "This is CD __ of __"

Description of Navigation

Type of DAISY DTB and how it can be used to be included in the script: "You can go directly to the sections in this DAISY book with a DAISY hardware or software player.

PLUS one of Option 1 through Option 7

  • Option 1: (repeated/almost identical or minimal textual headings that make word searching irrelevant, plus narrated audio) This DAISY book is produced with narrated audio divided into sections which represent the headings within the print book.
  • Option 2: (meaningful text headings, plus narrated audio) This DAISY book is produced with narrated audio and contains the text headings for each section. If you are using DAISY playback software on a computer you can also use a word search to find and go to the headings.
  • Option 3: (meaningful text headings and full text file of some portion/s of the book, plus narrated audio) This DAISY book is produced with narrated audio and contains the text headings for each section. In addition, the text of the Index/Glossary is present allowing you to do a word search in this portion of the book if you are using a DAISY software player on your computer.
  • Option 4: (meaningful text headings and full text file of some portion/s of the book, plus narrated audio; audio for text file is produced with synthetic speech) This DAISY book is produced with narrated audio and contains the text headings for each section. In addition, the text of the Index/Glossary is present allowing you to do a word search in this portion of the book if you are using a DAISY software player on your computer. The audio in this part of the book is produced with synthetic speech.
  • Option 5: (full text file of the book and narrated audio) This DAISY book is produced with narrated audio and contains the text file for the book. It is possible to do a word search on any word in the book if you are using a DAISY software player on your computer. The text will display on the computer screen as the audio is played.
  • Option 6: (full text file of the book and audio produced with synthetic speech) The audio in this DAISY book is produced with synthetic speech. It contains the text file for the book. It is possible to do a word search on any word in the book if you are using a DAISY software player on your computer. The text will display on the computer screen as the audio is played.
  • Option 7: (full text file only, no audio) This DAISY book contains the text file for the book. As there is no audio, this book must be read on a DAISY software player. It will be rendered in audio by the player or by the screen reader on your computer. It is possible to do a word search on any word in the book.

PLUS an explanation of what the various levels of headings represent

In this DAISY book:

  • Level 1 indicates: __________________________________________
  • Level 2 indicates: __________________________________________
  • Level 3 indicates: __________________________________________
  • Level 4 indicates: __________________________________________
  • Level 5 indicates: __________________________________________
  • Level 6 indicates: __________________________________________

Example:

  • Level 1 indicates: A Part, the Glossary and the Index
  • Level 2 indicates: Chapters
  • Level 3 indicates: Sections
  • Level 4 indicates: Subsections

Additional Navigation Features:

  • You can also navigate by phrase.
  • You can also navigate by page.
  • You can also navigate by page and phrase.
  • You can also navigate by page and paragraph.
  • You can also navigate by page, paragraph and sentence or phrase.
  • Page numbers are announced in the audio and are also present in the text display.
  • Page numbers are not included in the audio but they are present in the text display. When you search by page you will be taken to the beginning of that page, but the page number will not be announced in the audio. Players that have a "Where am I" function will read out the page number of the page you are reading when this function is selected.
  • Page numbers are "skippable". If your DAISY player supports skippable features, you can set it to play the audio for the page numbers or skip the page numbers.

Conclusion of Script:

This DAISY book begins on page ___.
